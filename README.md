# Midcycle Updates: Fisherman's Wharf Digital Menu Platform

This repository contains the full-stack architecture for the Fisherman's Wharf Digital Menu Platform, including the Next.js frontend and the hybrid AI agent backend.

## Repository Structure

The project is organized into three main components:

- **`menu/`**: The Next.js frontend for the digital menu platform.
- **`agent-ts/`**: The TypeScript-based agent developed using the Vertex AI Agent Development Kit (ADK).
- **`agent-python/`**: The Python-based Reasoning Engine agent, providing backend AI logic and specialized research capabilities.

## AI Agent Capabilities

The platform demonstrates advanced GenAI capabilities including:

- **🛡️ Customer AI Assistant**: Grounded in restaurant documentation via RAG and protected by Model Armor.
- **🎙️ Marketing Avatar**: High-fidelity digital human movement and speech indicating menu items.

## Getting Started

### Frontend (`menu/`)
Navigate to the `menu/` directory and run:
```bash
npm install
npm run dev
```

### TypeScript Agent (`agent-ts/`)
Navigate to the `agent-ts/` directory and run:
```bash
npm install
npm run dev
```

### Python Agent (`agent-python/`)
Navigate to the `agent-python/` directory and follow the Reasoning Engine setup instructions in its README.
